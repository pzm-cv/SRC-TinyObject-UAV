# SRC-TinyObject-UAV

Official repository for the paper:

**Selective Response Control Framework for Tiny Object Detection in Unmanned Aerial Vehicle Imagery**

This repository is created for code availability associated with the submitted manuscript. The full implementation, including model configuration files, training scripts, evaluation scripts, and usage instructions, will be released in this repository after the paper is accepted or officially published.

During the review stage, the code and trained models are available from the corresponding author upon reasonable request.

## Datasets

This work uses publicly available benchmark datasets:

* VisDrone2019-DET: https://github.com/VisDrone/VisDrone-Dataset
* AI-TOD: https://github.com/jwwangchn/AI-TOD
* DOTAv1.0: https://captain-whu.github.io/DOTA/

No new private dataset was created in this work.
